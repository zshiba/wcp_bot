@wcp_bot on Twitter

@wcp_bot is a Twitter bot tweeting weather by wcp commands.
@wcp_bot is using Twitter Streaming API to mominor wcp commands and Yahoo Weather API to obtain weather, and is working on Heroku.

Getting started:

STEP 1. Get your Twitter account followed by @wcp_bot

The best way to get it done would be you follow @wcp_bot.
This process is necessary to start using wcp command as @wcp_bot is monitoring the command on its timeline.

STEP 2. Tweet a wcpcommand toward @wcp_bot

The command Syntax is 

@wcp_bot wcp [-hat] LOCATION_WORDS [ZIP_CODE STATE COUNTRY ...]
where the options are:
-h: to show the help message,
-a: to make the reply from @wcp_bot anonymous, and
-t: to obtain weather for tomorrow.

For example, to @wcp_bot tweets the current weather in San Francisco CA USA, tweet the bellow with your Twitter account 

@wcp_bot wcp San Francisco CA USA

For example, to @wcp_bot tweets tomorrow's weather in Tokyo JAPAN, tweet the bellow with your Twitter account 

@wcp_bot wcp -t tokyo
("tokyo" could be written as "東京".)

Q&A

Q1. What if @wcp_bot guessed a wrong place?
A1. Give more location words, such as zip code, state, country name.

Q2. What if @wcp_bot did not respond to your wcp command?
A2. If your Twitter account appears to be protected, @wcp_bot cannot see your tweets publicly. In that case, please follow @wcp_bot first, and wait for @wcp_bot refollowing your Twitter account.

STEP 3. Done!

That's it! Any feedbacks would be appreciated.
